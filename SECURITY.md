# 🔒 Security Policy

## 📆 Supported Versions

We aim to keep **Netrilo Infrastructure** secure and up to date. The following versions are currently being supported with security updates:

| Version | Supported |
|---------|-----------|
| Latest  | ✅        |
| Older   | ❌        |

If you are using an older version, please consider upgrading to the latest release.

---

## 🐞 Reporting a Vulnerability

If you discover a security vulnerability in **Netrilo Infrastructure**, please report it responsibly.

### 📧 How to Report

- Create a private GitHub security advisory (via [Security Advisories](https://github.com/raminesfahani/Netrilo_Infrastructure/security/advisories)).
- Alternatively, email the maintainer directly if you'd prefer a private conversation.

Please **do not** disclose security issues publicly until they’ve been reviewed and patched.

### 🚨 What to Include

- A detailed description of the issue
- Steps to reproduce (if applicable)
- The potential impact or severity
- Any suggested fixes or workarounds

---

## ⏱️ Response Timeline

We aim to respond to security reports within **72 hours**, and will work with you to quickly investigate and address any issues.

---

## 🙏 Thank You

We appreciate responsible security disclosures that help keep the open source community safe.
